## セットアップ

1. このリポジトリをクローン
2. 必要に応じて `pubspec.yaml` の `assets` 設定や依存パッケージを `flutter pub get`
3. 画像は `img/screenshot/` 配下に入れてください
4. [campsite_go](https://github.com/ttsukahara967/campsite_go) サーバーを起動して、Flutterアプリを実行
